Welcome to List Comprehension Tool

ANTLR is a - as the name itself suggests "Another Tool for Language Recognition" developed by Terence Parr. I started using it a while ago and found it very intuitive to learn basics is language recognition and implementation. ANTLR is designed for both Java and .NET Framework. For reference please take a look at www.antlr.org.
I started this project not only to extend, but also to share my knowledge and skills in language recognition and implementation with ANTRL. Unfortunatelly there is not too much sources around, so I think this could be helpful on first place to myself but also to others, who intend to learn about these topics.
